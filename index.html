Setting up a HTML document:

Head Section
Object Code Purpose
!Use before head section! Doctype !DOCTYPE html Dictates that file is HTML
Metadata format meta charset="UTF-8" Sets the code to recognise specific characters
Metadata name meta name="description" content="Styling Notes"
meta name="author" content="John" 		Allows for page information to be set
Metadata name (recognising characters) meta name="description" content="ABC" Sets the code to recognise specific characters
Metadata name (Adapting for phones) meta name="viewport" content="width=device-width, initial-scale=1" Sets the code to change format when viewed on a phone
Stylesheet (local) link href='./style.css' rel='stylesheet' Connects a CSS sheet from your local folders
Stylesheet (outsourced) link href='https://www.codecademy.com/stylesheets/style.css' rel='stylesheet' Connects a CSS sheet from an online source
